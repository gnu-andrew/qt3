Menubar merging

This documentation is under development and is subject to change.

This example demonstrates the use of QMenuBar and QStatusBar in a QMainWindow to implement an in-place active control.

To build the example you must first build the QAxServer library. Then run qmake and your make tool in examples/menus.

The demonstration requires your WebBrowser to support ActiveX controls, and scripting to be enabled.
